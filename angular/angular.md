# ANGULAR

## ANGULAR CLI.

**Install**

```terminal
$ npm install -g @angular/cli
```

**Create a project**

```terminal
$ ng new <app-name>
```

**Create project (DRY RUN - Runs through the create process however no changes will be made to the file system)**

```terminal
ng new --dry-run
```

**Create a project (skip installing mode modules)**

```terminal
$ ng new flopbuster --skipInstall
```

**Run the application**

```terminal
$ ng serve --open / -o
```
