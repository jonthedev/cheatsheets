## INSTALL

```terminal
$ npm install truffle -g
```

## Initialize new project

```terminal
$ truffle init
```

**Contracts** - This is where you put your contracts.

**Migrations** - This tells truffle how to deploy your smart contracts.

**Test** - Test files locations

**truffle-config.js** - define deployment network for deploying your smart contract.

## Compile Project

```terminal
$ truffle compile
```

## Deploy Smart Contract Code

```terminal
$ truffle deploy --reset
```

## Launch Truffle Console

```terminal
$ truffle console
```
