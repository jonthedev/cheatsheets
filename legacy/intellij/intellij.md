# IntelliJ

## Run program

**NOTE**: These shortcuts may be different if vs code key bindings is DISABLED. Refer to Intellij documentation.

### VS CODE KEY BINDINGS ENABLED

```bash
cmd + shift + p   # open search window for all menu items
ctr + shift + r                     # run Program
cmd + n                             # Create a new file
cmd + click                         # step into class/method/etc
opt + tab                           # Goto Next Split Screen/Window
ctr + Enter                         # Generate classes/code
shift + enter                       # Open in right split
```

## INTELLIJ DEFAULTS

`navigation`

```bash
esc                                 # go back to editor
cmd + 1                             # Open project window/explorer
cmd + 4                             # Open run window
cmd + l                             # Go to line
cmd + click                         # step into class/method/etc
opt + tab                           # Cycle split Screen/Window
cmd + o                             # Go to classes
cmd + shift + [ / ]                 # Cycle tabs within split
cmd + fn + up / down                # Go to top / end of file
shift + cmd + a                     # open search window for all menu items
```

`edit`

```bash
ctr + g                             # find & select occurrences 1 by 1
ctr + cmd + f                       # Find all occurrences
cmd + opt + l                       # Format code
cmd + d                             # duplicate line
opt + shift + up / down             # move line up / down
md + opt + /                        # multi line comment
opt + opt(hold) + up / down         # multi carrot placement
opt + shift + click                 # multi carrot placement
cmd + shift + a                     # Toggle code wrap
opt + up                            # highlight sections of code
```

```bash
shift ( double tap)                 # Search Everywhere
ctr + r                             # run Program
cmd + n                             # Generate classes/code
cmd + w                             # close window
cmd + e                             # Show recent files
opt + enter                         # Show suggestions / **actions**
cmd + opt + v                       # variable suggestions
ctr + o                             # Select methods to override / implement
cmd + p                             # check parameters ()
ctl + space                         # Autocomplete Recommendation
```
