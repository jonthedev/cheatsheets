# JSHELL

## Basics

```bash
$ jshell                                # start jshell
$ /help                                 # help commands
$ /help keys                            # key shortcuts
$ /list                                 # show snippets of code
$ /edit <name or id>                # edit code snippets
$ ctr + r                               # search command history backwards
$ ctr + s                               # search command history forwards
$ ctr + a                               # move to the beginning of line (UNIX)
$ ctr + e                               # move to the end of line (UNIX)
$ ctr + u                               # delete to left of the cursor (UNIX)
$ ctr + c                               # abandon command
$ open/                                 # open a script
$ /set start -retain DEFAULT PRINTING   # set default scripts on jshell

```
