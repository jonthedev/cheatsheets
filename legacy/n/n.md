# n (node version manager)

[docs](https://www.npmjs.com/package/n#installation)

## Install

```terminal
$ npm install -g n
```

## Install node version

```terminal
$ n <version>
$ n lts (long term support)
```

## View downloaded versions and install selected

```terminal
$ n
```

## Remove node version

**Cached Versions**

```terminal
$ n <version> <another-version>
```

**All versions accept the installed version**

```terminal
$ n prune
```

## List downloaded versions in cache

```terminal
$ n ls
```

## Display diagnostics to help resolve problem

```terminal
$ n doctor
```
