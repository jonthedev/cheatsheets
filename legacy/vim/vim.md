# vim

---

`navigation`

```bash
j                           # down
k                           # up
l                           # right
h                           # left
ctr + f                     # move forward a page
ctr + b                     # move back a page
0                           # move to the beginning of the line
shift + 4                   # move to the end of the line
```

`edit`

```bash
dd                          # delete line
u                           # undo
x                           # delete including cursor position
shift + a                   # edit at the end of the line
shift + i                    # edit at the start of a line
```

`command mode`

```bash
shift + ;                   # enter command mode
0                           # go to top of file
shift + 4                   # go to end of file
```

`search`

```bash
/                           #enter search mode
```
