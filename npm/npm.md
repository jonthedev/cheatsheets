# NPM

# Install npm

```terminal
$ npm install install
```

# Check globally installed packages

```terminal
$ npm list -g
```

# See only top-level packages

```terminal
$ npm list -g --depth 0
```

# Removing a Global Package

```terminal
$ sudo npm uninstall -g <package-name>

```

# Update npm

```terminal
$ sudo npm update -g
```

# Update all global packages

```terminal
$ sudo npm update -g
```

# Update specific global package

```terminal
$ npm update -g <package_name>
```

## Save as a dev dependency

```terminal
$ npm install -D <package-name>
```

## Save as a dependency

```terminal
$ npm install -s <package-name>
```
