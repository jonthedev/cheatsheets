# NPM

# Check globally installed packages

```terminal
$ npm list -g
```

# See only top-level packages

```terminal
$ npm list -g --depth 0
```

# Removing a Global Package

```terminal
$ sudo npm uninstall -g <package-name>

```

# Update all global packages

```terminal
$ npm update -g
```

# Update specific global package

```terminal
$ npm update -g <package_name>
```
