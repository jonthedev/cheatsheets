# VS CODE

---

## Select a word and get all instances

- Shift + Cmd + L

## Select multiple lines

- Cmd + Option +Shift + UP/DOWN

## Select all occurrences of selected word in VSCode

- Cmd + shift + L

## Format Code

This one helps you adjust the code indentation:

- Shift + Option + F.

## Rename a Symbol

This is the quickest way to rename a symbol without searching the files and edit manually.

- F2

## Split Editor To The Right

Cmd + \

## Open VS Code settings

cmd + ,

## Turn on tab panel.

```terminal
$ cmd + ctrl + w
```

**Highlight the element/tag/component**
